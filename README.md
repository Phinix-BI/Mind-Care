# MindCare Website
MindCare is a web application designed to support mental health well-being through a combination of self-assessment, AI-driven assistance, and access to professional support. Our platform allows users to log in, complete a mental health assessment questionnaire, receive personalized results, engage in conversations with a trained AI model or therapist, and book appointments with healthcare professionals across India. ğŸ§
